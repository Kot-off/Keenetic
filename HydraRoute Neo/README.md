# Установка HydraRoute Neo

### ![Репозиторий](https://github.com/Ground-Zerro/HydraRoute)

## Установка

1. Выполните команду в терминале на вашем роутере:
opkg update && opkg install curl && curl -Ls "https://ground-zerro.github.io/release/keenetic/install-neo.sh" | sh

2. Веб админка hhtps://адрес_вашего_роутера:2000
(логин и пароль, как от вашей админки роутера)
